# Resala AUC Design System Package

Complete design system assets exported from the Canva Brand Kit.

## Package Contents

| File | Description |
|------|-------------|
| colors/brand-colors.css | CSS custom properties for all colors |
| colors/brand-colors.json | Color values in JSON |
| typography/brand-typography.css | CSS type scale |
| typography/brand-typography.json | Typography specs |
| tokens/brand-tokens.json | Combined design tokens |
| tokens/figma-tokens.json | W3C/Tokens Studio format |
| logos/README.md | Logo export instructions |
